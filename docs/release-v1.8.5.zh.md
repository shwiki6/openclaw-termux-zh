# OpenClaw Android 中文版 v1.8.5

独立 Android 应用，无需单独安装 Termux。

## 下载文件

| 文件 | 说明 |
|---|---|
| `OpenClaw-v1.8.5-arm64-v8a.apk` | 适用于大多数现代 Android 手机（推荐） |
| `OpenClaw-v1.8.5-armeabi-v7a.apk` | 适用于较老的 32 位 ARM 设备 |
| `OpenClaw-v1.8.5-x86_64.apk` | 适用于模拟器或 x86_64 设备 |
| `OpenClaw-v1.8.5-universal.apk` | 全架构通用包（体积更大） |
| `OpenClaw-v1.8.5.aab` | Android App Bundle（应用商店分发） |

## 首次运行

1. 安装 APK。
2. 按引导完成初始化（会下载约 500MB 的 Ubuntu RootFS）。
3. 在首页启动 Gateway。
4. 浏览器访问 `http://127.0.0.1:18789` 打开 Web 控制台。

## 系统要求

- Android 10+（API 29 及以上）
- 首次安装建议预留约 500MB 可用空间
- 首次初始化需联网
